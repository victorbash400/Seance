# Product Overview

This workspace contains two related projects:

## Seance
A Next.js web application featuring immersive 3D effects and interactive elements. The app includes:
- Spooky/mystical themed UI with smoke effects using react-smoke
- Interactive world map visualization with react-simple-maps
- Page-flipping book interface using react-pageflip
- 3D graphics powered by Three.js and React Three Fiber
- Multiple themed pages (mage, book) with atmospheric effects

## React-PageFlip
A React wrapper library for the StPageFlip library that creates realistic page-turning effects. This is a standalone npm package that provides:
- Realistic page flip animations for HTML content
- Mobile-friendly touch interactions
- Configurable animation timing and effects
- Support for both portrait and landscape orientations

The seance project uses react-pageflip as a dependency to create its book interface, demonstrating a real-world implementation of the library.
